# Sandbox_GruntTDD

Verifying how mature are the TDD frameworks to develop through TDD.

## Getting Started
Install the module with: `npm install Sandbox_GruntTDD`

```javascript
var Sandbox_GruntTDD = require('Sandbox_GruntTDD');
Sandbox_GruntTDD.awesome(); // "awesome"
```
